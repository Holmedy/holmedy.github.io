来自爱神之岛的引颈以待

发表于 2019-04-28 | 分类于 塞浦路斯

昨天惊喜地拿到了塞浦路斯的２欧元的硬币。之前想要是因为实在是太可爱了。

后来拿到以后查了查塞浦路斯的地理位置，不禁感慨它流通出来的不易。而且我实在怀疑本地是不是也已经很难拿到这种硬币了,因为我恰巧有个同学在塞浦路斯玩，我问她有没有看到塞浦路斯产的硬币，她说她看看，然后就没有下文了。。。

这个十字形偶来自公元前3000年，距今5000年的历史。这个岛上史前艺术的典型例子反映了塞浦路斯在古代文明的中心地位。

这个经典的雕塑又称珀莫斯人像，是一个张开双臂的女人，很可能被用作生育的标志，在塞浦路斯多次出现。小一点的可以作为附身符戴在颈上。最有趣的是，这个人像本身颈上戴着的就是个小号的它自己嘛！
幸好雕塑简陋到此结束，否则陷入无限的循环。也许这正是这些史前人民对于循环的思考，或者当妇女们戴上这个雕像的时候，是不是其本身也变成一个传承了这种生育意味的“珀莫斯”了呢？

你的名字

发表于 2019-04-06 | 分类于 法国

“Sur toutes les pages lues

Sur toutes les pages blanches

Pierre sang papier ou cendre

J’écris ton nom

在所有读过的书页上

在所有空白的书页上

石头、鲜血、白纸或灰烬上

我写下你的名字”

这浪漫，坚定，又热烈的诗句节选自Paul Eluard的《自由》的诗篇就这样一圈圈印在2015这枚纪念法国联盟节225周年的纪念币上，暗含着法国“自由”的座右铭。

毫无疑问，这菇凉还是法兰西女神玛丽安娜，戴着弗里吉亚帽。旁边这个同心圆的帽章其实是三色的（图片引用自 http://mes2eurocommemo.blogspot.com 这个作者做了好多好图，可能要经常引用了～＼（≧▽≦）／～）

一个冷的知识是1789年7月14日作为攻占巴士底狱的日子众所周知，但这个纪念的是1790年7月14日的联盟节。我觉得欧元里常常看到纪念带25周年这种也是非常可爱了！联盟节在巴黎战神广场举办，以纪念革命一周年。联盟节是大革命时期各政治派别最后一次团聚巴黎的活动。

法国领事馆对联盟节的解读说：“实际上，自1878年5月份，一位共和党人甘必大（Gambetta）的信徒就在一个有4000多人出席的大会上宣称：‘1789年7月14日，法国人民取得了自由。但是，哪里才是我们父辈伟大的光荣呢？是在1790年7月14日的战神广场（Champ-de-Mars）上。从那时起，法兰西民族建立起来了’。

1790年7月14日联盟节（Fete de la Federation）成了一个广泛的博爱运动的高潮。在巴黎，战神广场被改建造成一个宽阔的圆形阶梯剧场，剧场中央是竖立在土台上的祖国祭坛，约30万人参加了那次集会，其中有14000人是代表全国前来参加联邦节的外省代表。在祖国祭坛上举行的纪念弥撒结束之后，美国独立战争的英雄拉法耶特（La Fayette）以前来参加联邦节的代表的名义宣誓，法国人要相互团结起来，并和他们的国王一起捍卫自由、宪法和法律；随后，国王也宣誓遵守国民议会颁布的宪法。在外省，在市政府倡议下，所有居民聚集在一起，在同一时间，共同宣读了这个盟约。这样，公民们在一种同样的爱国主义热情中，接受了他们自己的节日。那时候，国家的统一不再是一种概念，而是一种实际的状况，但是，一些血腥措施，而首先就是1793年处死路易十六，使得这种统一在许多年中变得很脆弱。”

那个年代的风云激荡和狂热，让人感觉异常的遥远了，突然好想再看一遍《双城记》。

最后再分享一篇Paul Eluard的极其浪漫的诗：

　　　　　《宵禁》

　　大门被把守你说怎么办

　　我们被禁闭你说怎么办

　　交通被断绝你说怎么办

　　城市被管禁你说怎么办

　　全城在挨饿你说怎么办

　　我们被缴械你说怎么办

　　黑夜已沉沦你说怎么办

　　我们相爱了你说怎么办

这位大大现在长眠于拉雪兹公墓，下次去瞻仰一下。

自由，是你的名字。

日耳曼人的橡树情结

发表于 2019-03-05 | 分类于 德国

德国硬币分5个版本，与其他欧元硬币的区别是在硬币正面年份下面，会有A，D，F，G，J，5个不同的字母出现，其含义是代表由德国5个不同的铸币厂铸造，分别是A:柏林，D:慕尼黑，F:卡尔斯鲁，G:斯图加特，J:汉堡。

德国的１,2,5 欧分上的图样是橡树枝。橡树的原始产地是欧洲的葡萄牙，它属于寒带植被，在德国也有分布。由于不容易遭受病虫害，木材的用途也比较广泛，所以德国人都很喜欢它。受希腊神话的影响，德国人把它看作是一种神圣的树。

橡树又称金枝，是因为它在折下几月后，就会通体金黄。在现代人类学的奠基之作《金枝》中写道：“橡树在古代日耳曼人的宗教中，一直占有重要地位，雅格•格林说橡树是他们主要的神树。据说橡树是专门祭献给雷神道纳尔的圣树，而道纳尔就是意大利人的雷神朱庇特。基督教德意志总主教鲍尼菲斯在公元8世纪时，曾砍倒了盖斯玛附近的一棵圣橡，那棵树被信徒们称为“朱庇特的橡树”，在古日耳曼语中是‘道纳尔的橡树’。古希腊的罗马人奉橡树之神为雷神，认为他生育繁殖力强大，有降雨、保证大地丰收的神力，古日耳曼人也持这种看法。”在德国军队和政府中，很多勋章的设计都很精美，但是它们大多都和橡树有着直接的联系，而且很多陆军的旗帜上都带有橡树叶的图案。

橡树挺拔雄壮，据说其上的槲寄生也有神力。

季老回忆在哥廷根的留学经历时写到：“　古代的城墙仍然保留着，上面长满了参天的橡树。我在清华念书时，喜欢读德国短命抒情诗人荷尔德林的诗歌，他似乎非常喜欢橡树，可我始终不知道，橡树是什么样子。今天于无意中遇之，喜不自胜。此后，我常常到古城墙上来散步，在橡树的浓荫里，四面寂无人声，我一个人静坐沉思，成为哥廷根十年生活中最有诗意的一件事，至今忆念难忘。”

其实此图案还是基本沿用了德国马克时代的硬币芬尼背面的图案。

橡树在德国人心目中象征着不朽和顽强，特别是在浪漫主义时期，更是代表忠诚，也是德国精神的体现。要是哪位不善言辞的德国帅哥用橡树枝来表达自己如大海般沉静又汹涌的情感真是浪漫到要昏过去。而且在德国奥伊廷（Eutin）附近的森林还真的有一株特别有名的“新郎树”，人们向这棵有着500多岁的老树写信求爱，还真的就促成了不少的姻缘。
如果要寄信，当然是舒婷的《致橡树》最应景了，不知这位月老的中文好不好：

播种者

发表于 2019-02-22 | 分类于 法国

之所以把关于1,2,5欧分中出现的关于玛丽安娜的弗里吉亚帽子的小细节补齐，就是为了讲这个10,20,50欧分上出现的形象。这就是我说的，如果不是看这顶帽子，哪个知道这又是玛丽安娜啊。

一直以来播种者常作为劳动者的象征出现在许多绘画作品里。比如米勒画中那健壮有力，虎虎生风的形象，梵高也仿照这幅重画了一幅。

其实欧元硬币上的图样还是沿用了以前法郎的。

据说是当年设计者Roty从那画作播种者获得灵感，但将健壮的农民变成了苗条的玛丽安娜，带着弗里吉亚帽。这种模式引发了激烈的辩论。报纸反对它：“她播种的是什么，这个女人，带着花哨的弗里吉亚帽？她正在播下混乱，无政府状态，黑麦草，由谎言和不道德所滋生的仇恨。”（Le Moniteur，1897年2月28日）。然而，这个寓言很清楚。 “这些她慷慨播种的种子是无数的想法，在有一天我们不在了的时候，它们还会发芽。” （LaLiberté，1898年10月8日）。这种姿势实际上更具象征性而非现实性，因为人们不会将种子播放到风中。还有那身后半轮冉冉升起的朝阳，和射入天际的万道金光。

现今这个蜡的模型保存在奥赛博物馆里。

湖上的天鹅

发表于 2019-02-15 | 分类于 芬兰

芬兰是我最喜欢的西方国家。湖泊，森林，极光，圣诞老人童话一样的国度，养就了芬兰人礼貌，内敛，友善的性格，让我感觉一下子找到了同类。描述芬兰人这种内向有个笑话，“一个内向的芬兰人和你说话的时候看着自己的鞋子，一个外向的芬兰人和你说话的时候则看着你的鞋子”。我在国内也听过这个笑话，当时是喜程序员的。。。芬兰人这种含蓄的情感就可见一斑了。看着网上写芬兰人的那些个段子，我觉得每一件都是我会做的事。我只认识两个芬兰的朋友，但是我都非常喜欢他们，虽然我感觉他们一点都不内向。还记得在布拉格的冰吧，就是墙壁酒杯装饰都是冰做的那种，我们进去前都给我们发了防护装备，还检查我们是不是穿戴好了。我们这位芬兰的同学，还有一个挪威的小哥，就脱光了上身坐在那个也是冰做的王座上，大概是要展示身上流淌的斯堪的纳维亚血液，我们自然也是配以维京海盗似的欢呼和喝彩了。

芬兰人对于大天鹅有着独特的情感，这是他们的国鸟，是一种优美的、纯白色的北方野天鹅，显著的特征就是黄黑两色的喙。经常出现在芬兰民间传说和文化艺术作品中，在《卡勒瓦拉》民间史诗乃至作曲家西贝柳斯的作品中均有体现。西贝柳斯就这样写道：“今天10点到11点时，我看见了16只天鹅。我最棒的经历之一。尊敬的上帝，它们多美！它们环绕着我很久，并消失于日光薄雾之中，就像时隐时现的银缎带……天鹅的叫声很像铜管的声音。“三天后，他又写道：“天鹅总是出现在我的脑海，并给我的生命带来了壮丽的光彩……它们的声音存在于我的交响曲中……我的交响曲必将因此而显得与众不同。”

在芬兰商店内出售的许多产品上都印有一个飞翔的天鹅标志，表明这一产品符合北欧环保要求；或是印有一个游水的天鹅标志，表明这是在芬兰生产的食品。这枚1欧的硬币上则印有在芬兰典型的湖区风光中飞翔的一对大天鹅。

天鹅象征着坚韧和自由。当这种候鸟再度飞翔在芬兰的国土上时，正是春回大地的时节。也正因为如此，芬兰人总是将天鹅于光明，优雅和新的开端联系起来。又因为一对雌雄天鹅终生厮守，它们也象征着坚韧不拔的精神与矢志不渝的爱情。这大概是硬币上呈现一对天鹅的原因了。那我写这个原因也昭然若揭，新的一年，希望赐我光明，优雅，坚韧不拔的精神，和矢志不渝的爱情，当然还有Nokia.

百年沉思者罗丹

发表于 2019-02-09 | 分类于 法国

“生活不是缺少美，而是缺少发现美的眼睛。”
不知道我们小时候为什么这么喜欢引用这句话。这句话的作者，法国而著名雕塑家罗丹（Auguste Rodin) 已经去世一百多年了。2017年法国发行了纪念罗丹逝世一百周年的纪念币。

画面上他和他最有名的作品《思想者》前额对着前额。《思想者》当时被取名为《诗人》：它代表了《神曲》的作者但丁（Dante）。该作品启发了《地狱之门》的创作。但丁向前倾斜，为了观察地狱的九圈，并且思考自己的作品。《思想者》原本既是一个身体遭受虐待的人、近乎是一名下地狱者，又是一个具有自由意志的人，决心通过诗歌来化解苦难。

我们小时候流行一个“沉思者到底在思考什么”的笑话，答案是“到底要穿什么。”

“天才？绝对没有那种东西，有的只是用功、方法和不断地计划。”

仿佛看到山洞里的“原始人涂鸦”

发表于 2019-02-03

前面说有一款纪念欧洲经济货币联盟（EMU）成立十周年的设计让我忍俊不禁，前段时间终于被我收入囊中。 ^_^

让我回想当时的情景，大概是一边笑一边叹：“哇，这是真的硬币嘛！”
Hmmm，大概像连环画上滚铁环的孩子，墙上的儿童简笔画。于是我也忍不住拿起手边的记号笔，在白板上画了一个。。。
它表达了统一的货币，或者进一步说，欧洲经济货币联盟是欧洲长期贸易和经济一体化历史的最新一步。每个欧元区国家都发行了这一款硬币，并以相应的语言显示该国家的名称以及“EMU 1999-2009”。这款设计是由公众投票在五个候选名单中选出的，嗯看来大家和它的设计者，来自希腊的铸币部门的George Stamatopoulos，都是非常有幽默感滴人呢…

啊，转眼2009竟然是十年前了…废话不多说了，大家一起来笑一会吧！
